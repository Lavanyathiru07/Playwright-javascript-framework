Why Logger is static but Actions is instance-based?”
✅ Answer:

Logger is a utility that does not depend on any runtime context, so it is implemented using static methods. Actions and Assertions depend on the Playwright page
 object, so they are instance-based to maintain state and interact with the browser.

“Why Logger is static and not async?”
✅ Answer:

Logger is implemented as a static utility because it does not depend on any instance-level data. It performs synchronous operations like console logging, so there is no need for async/await. 
Async is used only where asynchronous operations like browser interactions or API calls are involved.

  “Why are hooks used instead of test.beforeAll?”
✅ Answer:

In Cucumber-based frameworks, we use Before and After hooks to manage the test lifecycle because 
Cucumber does not support Playwright’s test.beforeAll or fixture system. The Before hook ensures that 
each scenario starts with a fresh browser context, improving test isolation and reliability, while the 
After hook handles cleanup and failure reporting.


❓ “Why not use beforeAll?”
✅ Answer:

beforeAll runs once for all tests, which can cause state sharing and flaky behavior. In contrast, 
Cucumber’s Before hook runs before each scenario, ensuring proper isolation and making tests independent.

If interviewer asks:
👉 “Why can’t you define getByRole globally?”
✅ Answer:

Because Playwright locators like getByRole depend on the page instance, which is created at runtime. Defining them globally would result in undefined errors
 since the page is not available at that time.

import { selectors } from '../locators/selectors.js';
import Actions from '../utils/actions.js';

class SignupPage {

  constructor(page, context) {
    this.actions = new Actions(page, context);
  }

  async navigate(url) {
    await this.actions.navigate(url);
  }

  async signUp(name, email, password) {

    await this.actions.click(selectors.signup.signLoginBtn);

    await this.actions.fill(selectors.signup.name, name);

    await this.actions.fill(selectors.signup.email, email);

    await this.actions.fill(selectors.signup.password, password);

    await this.actions.click(selectors.signup.signupBtn);
  }
}

export default SignupPage;
